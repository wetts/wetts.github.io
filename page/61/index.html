jstl-c标签-out标签

发表于 2015-04-07 分类于 Java

<c:out>标签用来显示一个表达式的结果，与<%= %>作用相似，它们的区别就是<c:out>标签可以直接通过”.”操作符来访问属性。

举例来说，如果想要访问customer.address.street，只需要这样写：<c:out value=”customer.address.street”>。

<c:out>标签会自动忽略XML标记字符，所以它们不会被当做标签来处理。
