装饰器介绍

装饰器就是对被装饰的对象（函数、类）进行重构的，其可以在不改变原来对象的情况下调用对象时执行重构后的行为

  1. 解决问题：在函数执行之前和执行之后添加功能，调用函数的方式改变了
  2. 不改变原有函数的调用方法:函数里面嵌套函数，并且返回嵌套的函数
